High Limit System Tests
Test Specification Outcome
High Limit Temperature System: Activation

Set the oven High Limit setting to at least 10 degrees below the current chamber temperature and return to the oven display homepage. See the Set the High Temperature Limit procedure in the Operation chapter of the user manual.

When the High Limit cutoff activates:

  • A pair of flashing alert screens should alternate on the oven display.
  • A "4" indicator should turn on in the bottom display block.
  • The oven should cease heating.
 Does the High Limit System activate as described?
Yes No
   
High Limit Temperature System: Reset

Set the High Limit to 310°C, then press and hold the Reset button to clear the limit. Return to the temperature display to the homepage.

  • The alternating alert screens should cease flashing. The "4" indicator should turn off.
  • The oven should resume heating to reach the target set point of 150°C.

Note: Leave the High Limit system set to 310°C to prevent it from interfering with the Temperature Distribution Study. The High Limit may be set for normal operations once the Data Review Procedure is complete.

 Does the High Limit system turn off as described?
Yes No
