Temperature: Display Functionality

Inspect the temperature display on the unit. It should show a present temperature for the .

Note: When powered, the display will briefly show the current software build number, then show the temperature and target set point.

 Does the display show a temperature?
Yes No
   
Temperature: Control Functionality
  • Verify the temperature set point is set to 37°C. See the Set the Temperature Set Point procedure in the "Operation" chapter of the user manual. Set to 37°C if not already set.
  • The Heating Activated light should either remain on continuously or pulse as the incubator heats to 37°C.
  • Leave the temperature set to 37°C to prepare for the Temperature Distribution Study.
 Does the Heating Activated light illuminate?
Yes No
   
Temperature: Control Functionality

Set the temperature target setpoint to 150°C. See the Set the Temperature Setpoint procedure in the Operation chapter of the user manual.

  • After a brief pause, the temperature in the top, red display line should begin counting up. This indicates the oven is heating.
  • Leave the temperature set to 150°C to prepare for the uniformity study.
